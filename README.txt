В данной работе были применены следующие технологии:
	1. Библиотека ANTLR (https://www.antlr.org/) - утилита для распознавания языка, применяемая для парсинга текста по исходной грамматике;
	2. Попытка применения библиотеки GoDiagram (https://godiagram.com/avalonia/latest/index.html) - создание интерактивных диаграм для Avalonia UI. Впоследствии библиотека убрана из проекта, она еще на стадии prerelease и не проработана до конца.



Список литературы:
	1. How to create AST with ANTLR4? (https://stackoverflow.com/questions/29971097/how-to-create-ast-with-antlr4) - создание AST-дерева на основе парсинга ANTLR;
	2. Avalonia UI (https://docs.avaloniaui.net/) - документация по работе с фреймворком Avalonia UI;
	3. Avalonia Tutorial: Реализация MVVM по шагам с примерами (https://habr.com/ru/post/505036/) - статья на Хабр, дающая общее представление об Avalonia UI;
	4. ANTLR Online (http://lab.antlr.org/) - online-инструмент по работе с грамматиками ANTLR;
	5. Компиляция. 1: лексер (https://habr.com/ru/post/99162/), Компиляция. 2: грамматики (https://habr.com/ru/post/99298/), Компиляция. 3: бизон (https://habr.com/ru/post/99366/) - серия статей на Хабр по написанию собственного лексера (лексичесого анализатора) и парсера (синтаксический анализатор);
	6. Формальные грамматики и языки. Элементы теории трансляции (МГУ / учебное пособие для студентов 2-курса / И.А. Волкова, А.А. Вылиток, Т.В. Руденко) - пособие по теории формальных грамматик и языков (поверхностное ознакомление).
